# Fake Job Posting Detection

## 📌 Project Overview
This project is an AI-based Fake Job Posting Detection system that predicts whether a job posting is genuine or fake using Machine Learning.

## ✨ Features
- Detects fake and genuine job postings
- User-friendly web interface
- Confidence score prediction
- Fast and accurate classification

## 🛠 Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Streamlit
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- app.py
- train.py
- requirements.txt
- templates/
- static/
- job_model.pkl
- vectorizer.pkl

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/ahanaroychowdhury477/Fake-Job-Posting_Detection.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
streamlit run streamlit_app.py
```
